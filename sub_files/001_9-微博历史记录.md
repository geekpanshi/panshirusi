>
>> 1. 动议中心原则：动议是开会议事的基本单元。"动议者，行动的提议也。"会议讨论的内容应当是一系列明确的动议，它们必须是具体、明确、可操作的行动建议。先动议后讨论，无动议不讨论。
>> 2. 主持中立原则：会议"主持人"的基本职责是遵照规则来裁判并执行程序，尽可能不发表自己的意见，也不能对别人的发言表示倾向。（主持人若要发言，必须先授权他人临时代行主持之责，直到当前动议表决结束。）
>> 3. 机会均等原则：任何人发言前须示意主持人，得到其允许后方可发言。先举手者优先，但尚未对当前动议发过言者，优先于已发过言者。若对立严重，主持人应让意见相反的双方轮流得到发言机会，以保持平衡。

>> 5. 发言完整原则：不能打断别人的发言。
>> 6. 面对主持原则：发言要面对主持人，参会者之间不得直接辩论。
>> 7. 限时限次原则：每人每次发言的时间有限制 (比如约定不得超过10分钟）；每人对同一动议的发言次数也有限制（比如约定不得超过2次）。
>> 8. 一时一件原则：发言不得偏离当前待决的问题。只有在一个动议处理完毕后，才能引入或讨论另外一个动议。（主持人对跑题行为应予制止）
>> 9. 遵守裁判原则：主持人应制止违反议事规则的行为，这类行为者应立即接受主持人的裁判。
>> 10. 文明表达原则：不得进行人身攻击、不得质疑他人动机、习惯或偏好，辩论应就事论事，以当前待决问题为限。

>> 12. 多数裁决原则：（在简单多数通过的情况下）动议的通过要求"赞成方"的票数多于"反对方"的票数（平局即没通过）。弃权不计票数，表示支持所有结果，或者对结果没有意见。

>
> 投资，要多掌握一些常识，以至于不会被骗，或少犯些错误。
>
> 国人的“投资”热情，世界第一，也许是我们穷的太久了，总希望利用钱，再多赚些收益，满足一下匮乏，也不至于浪费钱这种资源。
>

>
>> 一、不要做劳模：总有人投资没赚到钱，或者亏损，认为是自己不够努力，专家也是这么说的，好像很少有人反思过，是不是投资的策略有问题。股市里卖来买去的人，多还跑不赢大盘，除了多缴税费，什么也做不到。楼市同理，来回折腾，税费巨大，还要面临诸多政策风险，得不偿失。
>> 通过个人努力，寻找到有价值的信号，实在是微乎其微，甚至可以说没有任何价值。如今，机器学习加大数据分析，人的努力完全无效，人工智能比对的数据量，与双腿努力看几套房子，更是两回事。打个比方，飞鸽传书再努力，远不如用微信，道理就这样简单。投资市场里，个人能做的不多，别折腾，至少，只亏不赚的事少干点，等于多赚了不少。
>>
>> 二、投资不是赌博：没有人可以逃顶和抄底，有也只是运气好而已。读到这里，会有人质疑，了解一下价格如何产生，你会豁然开朗。简单说，现在的价格，是由所有已知的信息决定的，未来的价格，是由未来的信息决定的，因此，价格不可以预测，因为，未来的信息还没有出现，也就没有价格涨跌可以产生，所有的预测皆是徒劳，没有任何可信度。如果非要把猜出来结果，当成了投资参考，无疑把投资变成了赌运气。
>>
>> 当然了，大趋势可以预期，如，经济增长、人口流入、货币供给量增加，都会导致稀缺资源价格上涨。价格上涨的快慢多少，取决于稀缺程度和人们对本商品的迫切性。这样看来，耍小聪明迟早会出问题，早买，差不多是唯一已知正确的选择。
>>

>> 5. 办理资金监管账户（成交价全部存在这个账户）；
>> 6. 评估公司对房屋评估（一般低评，成交价 80% 左右）；
>> 7. 和银行进行贷款面签（贷款以家庭为单位审批，可贷款额度按照评估价计算）；

>> 10. 放款后，办理二手房过户（交契税、个税、过户费等）；
>> 11. 过户后的过户单交给银行，等待银行最后审批；
